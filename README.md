# Student Result Portal

A web-based student result management system.

## Features

- Admin login system
- Create admin account
- Add student results
- Dynamic subject entry
- Edit and delete results
- Student login with roll number
- Result analytics charts

## Technologies Used

- HTML
- CSS
- JavaScript
- Chart.js

## How to Run

1. Download the project
2. Open index.html in browser
